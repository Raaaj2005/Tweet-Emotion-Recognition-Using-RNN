# 🐦 Tweet Emotion Recognition Using RNN

## 📖 Overview
This project delves into Natural Language Processing (NLP) and sentiment analysis by building a deep learning model to recognize emotions in Twitter data. The notebook walks through the entire NLP pipeline tailored for sequential text data, from importing and tokenizing raw tweets to applying padding and truncating, and finally training a Recurrent Neural Network (RNN) to accurately classify the underlying emotional tone.

## 🎯 Objective
The primary objective is to develop a robust text classification model capable of understanding the context and sequence of words in short-form text (tweets). By utilizing a Recurrent Neural Network, the project aims to demonstrate advanced text preprocessing techniques and the power of deep learning in extracting emotional sentiment from unstructured social media data.

## 🛠️ Tech Stack & Tools
* **Language:** Python
* **Environment:** Jupyter Notebook
* **Deep Learning Framework:** TensorFlow / Keras (RNN)
* **NLP Processing:** Keras Tokenizer, Sequence Padding
* **Data Manipulation & Visualization:** pandas, NumPy, Matplotlib

## 🧠 Key Learnings & Skills Demonstrated
* **Text Tokenization:** Converting raw string data from tweets into numerical tokens that a neural network can process.
* **Sequence Formatting:** Implementing padding and truncating techniques to ensure all text sequences have a uniform input length.
* **Label Preparation:** Properly encoding and preparing categorical emotion labels for the classification task.
* **RNN Architecture:** Designing and constructing a Recurrent Neural Network specifically optimized for sequential data and natural language.
* **Model Training & Evaluation:** Compiling the model, fitting it to the training dataset, and rigorously evaluating its emotion prediction accuracy on unseen test data.

## 🚀 Getting Started

### 📋 Prerequisites
To run this project, you will need:
* Python 3.x
* Jupyter Notebook or JupyterLab

### ⚙️ Installation & Setup

1. Clone the repository to your local machine:
   git clone https://github.com/Raaaj2005/Tweet-Emotion-Recognition-Using-RNN.git

2. Navigate into the downloaded project folder:
   cd Tweet-Emotion-Recognition-Using-RNN

3. Install the required dependencies:
   pip install tensorflow pandas numpy matplotlib jupyter

4. Launch the Jupyter Notebook environment:
   jupyter notebook

5. Open the project's .ipynb file and execute the cells sequentially to process the tweets and train the RNN model.
